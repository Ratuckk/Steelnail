# Hollow Knight: Steelnail  
A Hollow Knight fan-made fighting game built using **M.U.G.E.N**.

---

## 📂 Repository Purpose  
This repository exists solely to host the **character code files** for the game.  

If you are interested in playing the current version (**0.1**), you can download it here:

---

## ⚙️ About the Files (for M.U.G.E.N users)  
All files included for each character are currently **not indented or well-organized**.  
Because of this, making modifications may require experimentation.  
I plan to **rewrite and reorganize** these files starting with the addition of **Lace** to the game.

---

### 📑 File Overview
- **Nail.CNS** → Contains all universal states shared across characters.  
- **Helpers.CNS** → Organizes universal helpers for each character.  
- **Extra.CNS** → Holds states `-2` and `-3` (mostly universal).  
- **Normals.CNS** → Defines all normal attacks for the character.  
- **Projs.CNS** → Contains projectiles and character-specific helpers.
