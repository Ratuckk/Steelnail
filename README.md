### Sobre os arquivos (informação util apenas para quem tem experiência com mugen) 

#### Todos arquivos presentes em cada personagem colocado aqui pré-lace não estão identados e muito menos bem organizados, então é dificil criar suas próprias mudanças aqui sem muita experimentação, eu pretendo reescrever e ajeitar isso apartir da adição da lace no jogo.


Nail.CNS : É o arquivo de códigos onde há todos estados universais presentes em todos personagens.
Helpers.CNS : Destinado unicamente para organização dos helpers universais de cada personagem.
Extra.CNS : Arquivo dos estados -2 e -3 de cada personagem, a maioria é universal
Normals.CNS : É onde fica todos ataques do personagem.
Projs.CNS : É onde ficam especificamente os projéteis e helpers do personagem.
